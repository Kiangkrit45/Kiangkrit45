What is React.js

React.js is an open-source JavaScript library for building user interfaces, specifically for single-page applications. It's used for handling the view layer for web and mobile apps, hence allowing us to create reusable UI components.

Declarative > Imperative

React can improve the predictability and debugability of your code.

Imperative

HTML

<div id="counter">0</div>
<button id="incBin">inc</button>

JavaScript

let count = 0;
const incBin = document.getElementById("incBin");
const counter = document.getElementById("counter");
incBin.onclick = () => counter.textContent = ++count

Declarative

JSX

function Counter() {
    const [count, setCount] = useState(0)
    return {
        <>
            <div>{count}</div>
            <button onClick={() => setCount(count + 1)}>inc</button>
        </>
    }
}

Component-Based

Compose encapsulated components that handle their own state to create complicated user interfaces.

It's simple to transmit rich data through your app and keep state off of the DOM because component functionality is written in JavaScript rather than templates.

Write it anywhere you please

Because React.js makes no assumptions about the rest of your technological stack, you can use React to develop new features without having to rewrite old code.

React can also use Node (JavaScript runtime) to render on the server and React Native to power mobile apps.

A Basic Component

The render() method in React components takes in inputs and outputs what should be displayed.

This is how render() accesses input data that is supplied into the component via this.props.

JSX is not required to use React, but it is preferable to many users.

A Component With State

A component can store internal state data in addition to collecting input data (through this.props) (accessed via this.state). When the state data of a component changes, the rendered markup is changed by re-invoking render().

An Application

It's possible to create a Todo application using props and state. The state keeps track of the current list of items as well as the text provided by the user. Event handlers will be collected and implemented using event delegation, despite the fact that they appear to be rendered inline.

A constituent Using Third-Party Plugins

Using React to communicate with other libraries and frameworks. To convert the <textarea>, for example, remarkable, an external Markdown library, is used.

A Sample Project - Lotto Number Generator

npx create-react-app lotto

Now open your IDE and begin with JavaScript file, Lotto.js

import React from 'react'

function Lotto() {
    return {
        <>
            <h1>The three digit number is ...</h1>
            <div>1</div>
            <div>2</div>
            <div>3</div>
            <button>Random</button>
        </>
    }
}
export default Lotto

import React { useState } from 'react'

function randomNumber() {
    return Math.ceil(Math.random() * 9)
}

function Lotto() {
    const [counter, setCounter] = useState([0, 0, 0])
    const random = () => {
        setCounter([randomNumber(), randomNumber(), randomNumber()])
    }

    return {
        <>
            <h1>The three digit number is ...</h1>
            <div>{counter[0]}</div>
            <div>{counter[1]}</div>
            <div>{counter[2]}</div>
            <button onClick={random}>Random</button>
        </>
    }
}
export default Lotto
